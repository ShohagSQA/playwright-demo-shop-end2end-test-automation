Playwright Demo WebShop End to End Automation
This project automates the user registration and checkout flow for the Tricentis Demo WebShop using Playwright and JavaScript.

Project Description
The automation suite covers the complete user journey starting from account creation to order confirmation. It is designed to be stable, readable, and handles dynamic data to avoid test failures.

Features
User registration with unique email generation

Product navigation and selection for Desktops and Digital Downloads

Sorting products by price and adjusting page size

Shopping cart management and quantity updates

Complete checkout process including billing and shipping details

Automated order confirmation and verification

Tech Stack
Tool: Playwright

Language: JavaScript

Environment: Node.js

Installation and Setup
Clone the repository
git clone https://github.com/ShohagSQA/playwright-demo-shop-end2end-test-automation.git

Navigate to the directory
cd playwright-demo-shop-end2end-test-automation

Install dependencies
npm install

Install Playwright browsers
npx playwright install

Running Tests
Run all tests
npx playwright test

Run tests in headed mode
npx playwright test --headed

View test report
npx playwright show-report
